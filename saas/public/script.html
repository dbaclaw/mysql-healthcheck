单脚本、单 txt 输出，覆盖 13 大巡检模块的自动化数据采集工具
📋 脚本简介

mysqlHealthCheckV3.0.sh 是一款面向 MySQL / MariaDB 的专业巡检数据采集脚本，由 DBAClaw 维护。它通过单次执行即可完成操作系统、数据库实例、复制架构、存储对象、用户权限、会话锁、InnoDB 引擎、SQL 性能、日志、备份恢复、安全合规等 13 大模块的全面采集，输出标准化的 MySQLHealthCheck_<IP>_<时间戳>.txt 文件。

采集后的 txt 文件可直接上传到本平台的「MySQL 巡检报告生成器」，一键生成包含执行摘要、健康度评分、TOP SQL 治理、根因关联、合规对照等 17 章内容的商业可交付级 Word 报告。

✨ 核心特性

🔍 自动发现

从 ps -ef 自动解析 mysqld 进程参数，自动发现 basedir、socket、defaults-file、port 等关键配置

🧪 登录预检

采集前强制验证数据库连通性，支持 SSL 模式自动降级，避免生成空报告

📦 单文件输出

所有 13 个模块的数据统一输出到单个 txt，便于传输、归档和后续自动化处理

⚙️ 灵活配置

支持命令行参数、交互式补全、模块跳过（--skip-modules）、自定义备份路径等

🛡️ 安全脱敏

自动对 my.cnf 中的 password、ssl_key 等敏感字段进行掩码处理（*******）

📊 版本兼容

完美兼容 MySQL 5.6 / 5.7 / 8.0 及 MariaDB，自动识别版本并切换对应 SQL 方言

🚀 快速开始

1. 交互式用法（推荐首次使用）：

./mysqlHealthCheckV3.0.sh

脚本会自动发现运行中的 mysqld 进程参数，并提示你确认/补全连接信息。

2. 命令行参数用法（适合批量/自动化）：

./mysqlHealthCheckV3.0.sh \
  --user dbadmin --password 'YourPass' \
  --host 127.0.0.1 --port 3306 \
  --output-dir ./reports \
  --slow-log-lines 5000 \
  --error-log-lines 1000

3. 仅测试登录：

./mysqlHealthCheckV3.0.sh --test-login --non-interactive

4. 跳过特定模块：

./mysqlHealthCheckV3.0.sh --skip-modules "logs,backup,interview"
